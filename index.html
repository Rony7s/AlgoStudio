Bubble Sort

Bubble sort is a basic sorting algorithm that iteratively compares adjacent elements in a list and swaps them if they are in the wrong order. This process continues until no more swaps are needed, resulting in a sorted list. It's named "bubble sort" because larger elements tend to bubble their way to the end of the list during each pass. While bubble sort is straightforward to understand and implement, it is not efficient for large datasets due to its time complexity of O(n^2), where n is the number of elements. As a result, it's primarily used for educational purposes or small datasets, and more efficient sorting algorithms are favored in practical applications.


                    
#include<iostream>
using namespace std;

int main(){
    int n;
    cout <<"Enter Array Size: "<<endl;
    cin >>n;
    int arr[n];

    for(int i=0; i<n; i++){
        cout <<"Enter Array element of index: "<< i <<endl;
        cin>>arr[i];
    }
    //bubble Sort
    for(int i=0; i<n; i++){
        for(int j=0; j<n-i-1; j++){
            if(arr[j]>arr[j+1]){
                //swap two numbers without using third variable
                arr[j] = arr[j] + arr[j+1];
                arr[j+1] = arr[j] - arr[j+1];
                arr[j] = arr[j] - arr[j+1];
            }
        }
    }

    cout <<"The Sort result is : "<<endl;
    for(int i=0; i<n; i++){
        cout << arr[i]<<" ";
    }

    return 0;
}

Bubble Sort Algorithm:


Start with an unsorted array of elements.
Repeat the following steps until no more swaps are needed:
a. Compare adjacent elements in the array, starting from the first element (index 0).
b. If the current element is greater than the next element, swap them.
c. Move to the next pair of adjacent elements and repeat the comparison and swapping process.
After completing a full pass through the array, the largest element will have "bubbled up" to the end of the array (i.e., it will be in its correct sorted position).
Repeat steps 2 and 3 for the remaining unsorted portion of the array (excluding the last element, which is already sorted).
Continue these steps until you complete a pass through the entire array without any swaps. This indicates that the array is now fully sorted.



Advantages of Bubble Sort Algorithm

Bubble sort is a simple sorting algorithm with some advantages:

  • Simplicity: Bubble sort is one of the simplest sorting algorithms to understand and implement. It involves comparing adjacent elements and swapping them if they are in the wrong order. This simplicity makes it a good choice for educational purposes or as a starting point for learning about sorting algorithms.
  • No additional data structures: Bubble sort doesn't require any additional data structures such as arrays or linked lists for sorting. It works directly on the given array, which can be an advantage when memory usage is a concern.
  • In-place sorting: Bubble sort is an in-place sorting algorithm, meaning it doesn't require additional memory for temporary storage of elements. This can be advantageous when working with limited memory resources.

However, it's important to note that bubble sort has several significant disadvantages, including its poor time complexity. It is not suitable for sorting large datasets because it has a time complexity of O(n^2), where n is the number of elements to be sorted. There are much more efficient sorting algorithms available, such as quicksort, mergesort, and heapsort, which are generally preferred for practical applications when performance is a concern. Bubble sort is mainly used for educational purposes or when simplicity is more important than efficiency.
